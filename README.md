# Login Google Project

Este proyecto incluye:
- 📱 App móvil en React Native (Android/iOS)
- 💻 Web responsive en React + Vite + TypeScript
- 📦 APK en carpeta `apk`

## Cómo iniciar

### Web
```
cd web
npm install
npm run dev
```

### Mobile
```
cd mobile
npm install
npm run android
```

El APK se generará en la carpeta `apk` tras compilar en Android Studio.
